## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Build the APL JSON document for a horizontal carousel of recently played items. Each item shows a primary image (poster/album art), title, and subtitle (artist/album or season/episode). Items are touchable — tapping sends a `UserEvent` with the item's Jellyfin ID.

**Implementation notes:**
- Add to `AplHelper.cs` alongside existing `BuildNowPlayingDocument()` and `BuildListDocument()`
- Use APL `Carousel` component or `Sequence` with `horizontal` scrollDirection
- Each item uses `TouchableWrapper` with `onPress` sending a `UserEvent` request with the item ID as a token
- Image URLs come from `BaseHandler.GetImageUrl()` pattern: `Items/{id}/Images/Primary?api_key={token}`
- Include a fallback text-only hint for devices that partially support APL
- Handle empty state: show a "What would you like to hear?" prompt with the skill's branding
- The carousel should include a header like "Recently played" or use the locale string system

**Depends on:** nothing (standalone template)
