## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Wire the reusable APL list template into `SearchMediaIntentHandler` for disambiguation responses.

**Context**: When a search returns multiple matches (e.g., "I found 3 matches: song X, album Y, artist Z"), the handler asks the user to clarify. On Echo Show, these options should be displayed as a tappable list.

**Implementation**:
1. When disambiguation occurs (multiple results), build a `ListDisplayItem` list from the matches
2. Check `DeviceSupportsApl()` and add `BuildListDirective()` to the response
3. Map each search result to ListDisplayItem with appropriate Title, Subtitle (content type), ArtUrl, and Id
4. Handle `UserEvent` from tap to route to the correct match selection

**Special consideration**: Search results can be mixed types (Audio, MusicAlbum, Artist, Movie, Series). The APL subtitle should indicate the content type (e.g., "Song", "Album", "Artist") to help disambiguation.
<!-- SECTION:DESCRIPTION:END -->
