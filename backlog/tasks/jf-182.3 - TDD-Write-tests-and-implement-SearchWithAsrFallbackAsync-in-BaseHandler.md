## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Add a `SearchWithAsrFallbackAsync()` method to `BaseHandler` that wraps the search-retry logic with the ASR compound word fix.

## Method signature
```csharp
protected async Task<IReadOnlyList<T>> SearchWithAsrFallbackAsync<T>(
    string query,
    Func<string, Task<IReadOnlyList<T>>> searchFunc,
    Request request)
```

## Behavior
1. Always call `searchFunc(query)` first (original query)
2. If results.Count > 0 → return results (no extra work)
3. Check `AsrCompoundWordFixEnabled` from config — if false, return empty results
4. Generate variants via `AsrVariantGenerator.GenerateAsrVariants(query)`
5. For each variant, call `searchFunc(variant)` — stop on first non-empty result
6. Return results if found, otherwise return empty list

## TDD
Write tests FIRST in `BaseHandlerAsrFallbackTests.cs`. Mock the search func to track calls and control results. Test both feature-on and feature-off paths. Then implement.
