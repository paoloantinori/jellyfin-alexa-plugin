## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Create an intent handler that processes APL `UserEvent` requests triggered by tapping a carousel item. The handler extracts the Jellyfin item ID from the event token, looks up the item, and starts playback.

**Implementation notes:**
- Alexa sends `Alexa.Presentation.APL.UserEvent` requests when a TouchableWrapper is pressed
- The handler's `CanHandle()` checks for `request is UserEventRequest` with the matching APL interface
- Extract item ID from the event's `arguments` or `source.token` (set by the carousel template in JF-176.1)
- Load the item from `ILibraryManager.GetItemById()` or `GetItemList()` with the ID
- If audio → `BuildAudioPlayerResponse()`. If video → `VideoAppLaunchDirective`.
- Add locale strings for error fallback
- Register in the pipeline (check `RequestPipeline.cs` for handler registration)

**Depends on:** JF-176.1 (APL template sets the token format)
