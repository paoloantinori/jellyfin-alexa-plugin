## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Add an APL detail card to `MediaInfoIntentHandler` for displaying information about the currently playing track on Echo Show.

**Context**: MediaInfoIntentHandler returns various types of information (artist bio, genre, duration, etc.) as speech. On Echo Show, this should render as a detail card with album art and structured text.

**Implementation**:
1. Create a `BuildDetailDirective()` method in `AplHelper` (or reuse existing `BuildNowPlayingDirective` with modified data)
2. Show: album art, track title, artist, and the specific info requested (genre, duration, etc.)
3. Check `DeviceSupportsApl()` before adding the directive

**Note**: This is lower priority than the list handlers since MediaInfo is a single-item response. The existing Now Playing template could potentially be reused with the info text as subtitle.
