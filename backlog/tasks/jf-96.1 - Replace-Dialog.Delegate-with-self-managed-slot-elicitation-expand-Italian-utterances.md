## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Fix the broken Dialog.Delegate calls that crash on it-IT locale (no dialog model exists), and add more Italian carrier phrases to improve intent-matching confidence for English proper nouns.

**Problem**: Handlers in `Alexa/Handler/Intent/PlaySongIntentHandler.cs`, `PlayAlbumIntentHandler.cs`, `PlayEpisodeIntentHandler.cs`, and `PlayPodcastIntentHandler.cs` call `DelegateToDialog()` when `DialogState != Completed`. This returns a `Dialog.Delegate` directive which Alexa rejects because:
1. The skill manifest (`Alexa/Manifest/manifest.json`) lacks the Dialog interface
2. Only en-US has dialog models — it-IT and 10 other locales only have `languageModel`

**Solution approach**:
1. Replace `DelegateToDialog()` with self-managed slot elicitation using `ResponseBuilder.Ask()` with locale-specific prompt strings
2. Add Italian prompt strings to `Alexa/Locale/it-IT.json` (e.g., `ElicitSongName`, `ElicitAlbumName`)
3. Add 15-20 more Italian carrier phrase variations to `PlaySongIntent` in `model_it-IT.json`:
   - `"suona {song}"`
   - `"metti {song}"`
   - `"fai partire {song}"`
   - `"riproduci {song} di {musician}"`
   - `"ascolta {song}"`
   - `"suona il brano {song}"`
   - `"metti la canzone {song}"`
   - etc.
4. Add similar expansions for `PlayAlbumIntent` and `PlayArtistSongsIntent`
5. Add concrete (non-slotted) utterances for NLU disambiguation

**Files to modify**:
- `Jellyfin.Plugin.AlexaSkill/Alexa/Handler/BaseHandler.cs` — remove or guard `DelegateToDialog`
- `Jellyfin.Plugin.AlexaSkill/Alexa/Handler/Intent/PlaySongIntentHandler.cs` — self-managed elicitation
- `Jellyfin.Plugin.AlexaSkill/Alexa/Handler/Intent/PlayAlbumIntentHandler.cs` — same
- `Jellyfin.Plugin.AlexaSkill/Alexa/Handler/Intent/PlayEpisodeIntentHandler.cs` — same
- `Jellyfin.Plugin.AlexaSkill/Alexa/Handler/Intent/PlayPodcastIntentHandler.cs` — same
- `Jellyfin.Plugin.AlexaSkill/Alexa/InteractionModel/model_it-IT.json` — more samples
- `Jellyfin.Plugin.AlexaSkill/Alexa/Locale/it-IT.json` — elicitation prompt strings
- `Jellyfin.Plugin.AlexaSkill/Alexa/Locale/ResponseStrings.cs` — new string keys
- `tests/integration/fixtures/it-IT.yaml` — new test cases with expected slot values
