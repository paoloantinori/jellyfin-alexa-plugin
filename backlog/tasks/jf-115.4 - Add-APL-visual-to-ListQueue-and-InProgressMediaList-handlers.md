## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Wire the reusable APL list template into `ListQueueIntentHandler` and `InProgressMediaListIntentHandler`.

**ListQueueIntentHandler**: Lists upcoming queue items. Note that `AplHelper.BuildQueueDirective()` already exists for queue display — consider whether to reuse it directly or use the new generic `BuildListDirective()`. The existing queue template has tap-to-play, which is ideal.

**InProgressMediaListIntentHandler**: Returns up to 5 in-progress items with playback positions (e.g., "Movie (1h 30m remaining)"). APL list should show thumbnails with progress text.

**Implementation pattern** (same as other handlers):
1. Check `DeviceSupportsApl(context)` 
2. Build `ListDisplayItem` list from handler results
3. Add `BuildListDirective()` to response directives
4. Handle tap events for item selection

**For ListQueue**: Consider reusing the existing `QueueDocument` APL template since it already has track-specific layout. The new `BuildListDirective` could be used if the queue items are a simple list.
<!-- SECTION:DESCRIPTION:END -->
