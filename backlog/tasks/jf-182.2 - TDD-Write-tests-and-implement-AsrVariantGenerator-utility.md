## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Create a static utility method `GenerateAsrVariants(string query)` that produces the search variants for a given ASR query. This is a pure function with no dependencies — easy to test in isolation.

## Method signature
```csharp
public static IReadOnlyList<string> GenerateAsrVariants(string? query)
```

## Algorithm
1. If query is null/empty/whitespace or has ≤1 word → return empty list
2. Generate N-1 pairwise joins (join each adjacent word pair, keep the rest as-is)
3. Add the fully-collapsed variant (all spaces removed) if not already in the list
4. Return deduplicated list

## TDD
Write ALL unit tests FIRST in a new test file `AsrVariantGeneratorTests.cs`. Run them — they should all fail. Then implement the method to make them pass.

The method should live in a new utility class `Alexa/Util/AsrVariantGenerator.cs` as a pure static method with no Jellyfin dependencies.
